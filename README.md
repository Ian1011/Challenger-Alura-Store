# Challenger Alura - Alura Store
## Objetivo do Projeto
Este projeto faz parte de uma série de desafios da Alura na trilha Data Science do programa Oracle, esse desafio se trata de uma avaliação de 4 lojas para determinar qual é a melhor opção a ser vendida para um novo investimento, para esta escolha é preciso uma análise nos dados do desempenho de cada loja.

## Dados analisados
A análise foi feita com base nas seguintes métricas:
* Faturamento total de cada loja
* Avaliação Média
* Frete Médio
* Vendas por categorias
* Produtos mais e menos vendidos
